# Portfolio & Quiz Web App

## Overview
This project is a personal portfolio combined with a fully functional Quiz Web App.  
It is built using **HTML**, **CSS (Flexbox)**, and **JavaScript (DOM + Events)**.  

The homepage showcases the portfolio and provides a link to the live Quiz App.  
The Quiz App features MCQs and True/False questions with **live scoring**, **immediate feedback**, and **random background color changes** on load and window resize.

---

## Features

### Homepage
- Portfolio introduction with description
- Button linking to the live portfolio
- Button linking to the Quiz App
- Responsive design using Flexbox
- Gradient and stylish buttons

### Quiz App
- Multiple-choice and True/False questions
- Live score tracking
- Immediate feedback for each answer
- "Next", "Quit", "Restart", and "Home" buttons fully functional
- Random background color on load and on window resize
- Fully responsive design for mobile and desktop

## Usage

- Open `index.html` to access the homepage.  
- Click **Click Here to Access Quiz App** to start the quiz.  
- Answer the questions and see live feedback and scoring.  
- Click **Play Again** to restart or **Home** to go back to the homepage.
